Este projeto foi desenvolvido para Imersão Dev com Google Gemini 10ª Edição
O projeto usa como base o liberado durante a imersão, o mesmo foi modificado para concorrer a premiação
O projeto desenvolvido consiste em uma pagina que traz centenas de receitas culinarias
As receitas foram geradas usando o gerador com google gemini, sendo modificado o imput e a base para atender o projeto
Das funções do projeto, no campo de pesquisa, é possivel pesquisar não só o nome da receita, mas por igredientes e equipamentos, trazendo somente as receitas que possuem esses itens e o nome na receita
